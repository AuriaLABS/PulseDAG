## Archive policy

- Historical evidence is never relabelled as v2.3.0 evidence.
- Superseded operator guidance is removed from active indexes.
- Exact content remains available through immutable commits and tags.
- A historical helper may remain in an active path only when a current workflow still invokes it and its compatibility role is documented.
- New v2.2.x files are forbidden outside archive/evidence paths.
